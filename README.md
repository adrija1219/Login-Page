# Project Overview

This project implements a modal-based login and registration system with animated transitions between forms.
It is designed to showcase UI state management, CSS animations, and responsive layout techniques without relying on external JavaScript frameworks.

## Key Features

Modal-based Login & Registration system

Smooth CSS sliding animations between forms

Glassmorphism UI using backdrop-filter and transparency

Responsive design using Flexbox

Reusable and maintainable code structure

Icon-based inputs using IonIcons

Clean, modern typography using Google Fonts (Poppins)
